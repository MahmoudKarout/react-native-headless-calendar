# react-native-fast-calendar

A headless, calendar-system-agnostic, fully composable React Native calendar. Bring your own UI primitives, your own theme, your own calendar system (Gregorian, Hijri, Chinese, Ethiopian, …).

## Installation


```sh
npm install react-native-fast-calendar
```


## Usage


```js
import { multiply } from 'react-native-fast-calendar';

// ...

const result = multiply(3, 7);
```
